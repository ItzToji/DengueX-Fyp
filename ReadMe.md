DengueX is an AI-assisted digital health platform designed to monitor, analyze, and help prevent dengue outbreaks through real-time data visualization, intelligent health guidance, and citizen participation.

The system integrates secure user management, interactive dashboards, AI-based mosquito detection, and an AI health assistant to support both citizens and public health authorities in early awareness and informed decision-making.

This project was developed as part of a Final Year Project (FYP) in Computer Science / Software Engineering.
